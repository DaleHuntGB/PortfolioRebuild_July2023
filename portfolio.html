Game Engine Sub-System Development

This was a university assignment where we provided a Game Engine created using C++ and SDL. Our instruction was to create a sub-system and due to my interest in user-interface, I chose to create a functional pause/start menu from scratch.

The goal was to create a general user-interface that could be utilized by any game that may be using this Game Engine, to my knowledge, all games require a Pause/Start Menu and therefore, I thought this would be a very functional sub-system.

If you are interested, I have a very detailed demonstration of how the sub-system works. I explain incorporation, adding buttons, editing buttons and overall my experience.
